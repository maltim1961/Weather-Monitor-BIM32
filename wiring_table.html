📌 Важные примечания

1. Конфигурация TFT_eSPI: Пины дисплея ILI9341 задаются в файле User_Setup.h библиотеки TFT_eSPI, а не в коде проекта.

2. I2C шина: Все I2C устройства подключены параллельно к GPIO 21 (SDA) и GPIO 22 (SCL). Необходимы подтягивающие резисторы 4.7 кОм к 3.3В на каждой линии.

3. GPIO 34–39: Эти пины только на вход (input only), не имеют внутренних подтяжек. При использовании кнопок нужны внешние подтягивающие резисторы.

4. Nextion vs ILI9341: Используется только один тип дисплея за раз. При использовании ILI9341 пины 32/33 для Nextion не заняты (GPIO 33 используется для WS2812B).

5. DFPlayer: Управление осуществляется через Serial (UART0) с динамическим переключением скорости (115200 ↔ 9600 бод).

6. Partition Scheme: No OTA (2MB APP / 2MB SPIFFS/LittleFS).
